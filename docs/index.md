# 探索与折腾：我们的 ERPNext 实践笔记

欢迎来到这里！这不仅是一个开源项目库，更是一个充满了尝试、踩坑和分享的极客空间。

我们是一群对企业数字化、开源 ERP 系统感兴趣的玩家和独立开发者。平时喜欢自己动手写代码、搭服务器，顺手通过这些开源工具来助力一下自己小微团队的日常运营。

## 我们的初衷
我们希望通过记录和分享在 Frappe / ERPNext 这个强大的低代码平台上的实战经验，把那些晦涩的官方文档变成接地气的大白话。让更多感兴趣的开发者或者创业者，能以极低的成本，搭建出属于自己的好玩又好用的数字后花园。
